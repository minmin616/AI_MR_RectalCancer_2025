## Overview
This repository contains the **fully reproducible R code and raw plotting datasets** used in:  
**"AI-Enabled Dynamic Magnetic Resonance Imaging for Rectal Cancer Staging and Treatment Planning"** .  

Three prospective phases were conducted:
1. **Phase 1** – protocol optimization (224 patients with paired conventional and dynamic MRIs)  
2. **Phase 2** – multi-reader multi-case validation (106 patients with paired conventional and DynaFusion MRIs)  
3. **Phase 3** – clinical-impact cohort (98 patients with paired conventional and DynaFusion MRIs)
