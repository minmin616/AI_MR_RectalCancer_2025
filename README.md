# Live-Fusion MRI for Rectal Cancer Staging  
&gt; R-scripts & plotting data for **Phase 1 / 2 / 3** analyses (R ≥ 4.3.2)

---

## Overview
This repository contains the **fully reproducible R code and raw plotting datasets** used in:  
**"AI-enabled MR live imaging in rectal cancer: a prospective, step-wise evaluation on clinical application"** .  

Three prospective phases were conducted:
1. **Phase 1** – protocol optimisation (*n*=224 paired datasets)  
2. **Phase 2** – multi-reader multi-case validation (*n*=106 paired datasets, 23 readers)  
3. **Phase 3** – clinical-impact cohort (*n*=98 paired datasets)
